# BudgetBuddy


BudgetBuddy is een simpele web-app om je **inkomsten** en **uitgaven** bij te houden.  
Je kunt transacties toevoegen, verwijderen en je saldo zien. Alles wordt opgeslagen in je browser (localStorage).

## Functies
- Inkomsten en uitgaven toevoegen
- Overzicht: totaal inkomsten, totaal uitgaven en saldo
- Transacties bekijken en verwijderen
- Filteren op: alles / inkomsten / uitgaven
- Opslaan in de browser (blijft na refresh)
- Export naar JSON

## Hoe start je dit project (lokaal)

### Optie 1: Met Live Server (aanrader)
1. Open de map in **Visual Studio Code**
2. Installeer de extensie **Live Server**
3. Rechtsklik op `index.html` → **Open with Live Server**
4. De app opent in je browser

### Optie 2: Zonder Live Server
- Dubbelklik op `index.html` en het opent in je browser.

## Bestanden
- `index.html` = de pagina (layout/structuur)
- `styles.css` = styling (hoe het eruit ziet)
- `app.js` = logica (toevoegen, berekenen, opslaan)

## Opslag
De app gebruikt **localStorage**.  
Dat betekent: je data wordt opgeslagen in je browser op jouw apparaat.

## GitHub Pages (online zetten)
1. Ga naar je repo op GitHub
2. **Settings** → **Pages**
3. Kies: **Deploy from a branch**
4. Branch: `main` en folder: `/ (root)`
5. Save → je krijgt een link naar je website
